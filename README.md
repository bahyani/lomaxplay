## ğŸ¯ User Flows

### ğŸ† Tournament Competition Flow

1. **Visit Homepage** - Navigate to `https://www.lomaxplay.com`
2. **Create Account** - Register with email and complete profile setup
3. **Browse Tournaments** - Explore available competitions by game type
4. **Tournament Registration** - Select and register for desired tournaments
5. **Pay Entry Fee** - Complete one-time tournament entry payment
6. **Compete** - View bracket placement and participate in matches
7. **Track Performance** - Monitor statistics, rankings, and tournament history

### ğŸ’» Software Rental Flow

#### As a Renter (Using Software)
1. **Visit Homepage** - Navigate to `https://www.lomaxplay.com`
2. **Create Account** - Register and verify your profile
3. **Browse Software** - Explore available gaming software catalog
4. **Select Software** - Choose software with rental period (daily/weekly/monthly)
5. **Make Payment** - Pay rental fee for selected duration
6. **Access Software** - Receive download link or access credentials
7. **Use Software** - Utilize rented software within rental period

#### As a Lister (Offering Software)
1. **Visit Homepage** - Navigate to `https://www.lomaxplay.com`
2. **Create Account** - Register and complete verification
3. **Update Collection** - Add your owned gaming software to profile
4. **List for Rent** - Post software with rental pricing and terms
5. **Manage Listings** - Update availability and pricing
6. **Receive Rentals** - Get notified when users rent your software
7. **Earn Revenue** - Receive payment credits to your wallet
8. **Track Earnings** - Monitor rental income and transaction history
